# Terminal Video Player
> `Usage: player [options] <source_file>`

There is no options yet.

Translates video into characters: `' .:-=+*#%@'` (darker -> lighter). Based on opencv lib.

Key | Action|
:---:|:---:
`q` | `quit`
`' '` | `pause/play`

## Notes
- cant play sound yet
- opencv should be installed
